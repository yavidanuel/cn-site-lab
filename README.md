# cn-site-lab

## 项目简介

cn-site-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常用于存放一些静态的、非商业化的实验性内容、个人笔记或轻量级工具页面。本仓库不对应任何特定网站或域名，仅作为 HTML 文件的集中存储与展示空间。

## 目录说明

```
cn-site-lab/
├── pages/          # 存放所有独立 HTML 页面，每个页面一个文件或子目录
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── LICENSE         # 开源许可证
```

- `pages/`：该目录下每个子文件夹或独立 HTML 文件代表一个独立页面。页面之间无关联，可单独访问。
- `assets/`：页面共用或专用的资源文件，按需组织存放。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，不依赖外部数据库或后端服务。
- 页面内容可能包含实验性功能、学习笔记、样式测试等，不保证长期更新或兼容所有浏览器。
- 文件命名尽量使用英文小写和短横线分隔，便于 URL 访问。
- 部分页面可能引用外部资源（如 CDN 库），请确保网络可达。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，但请确保内容不违反法律法规及平台规定。
- 如需修改已有页面，请说明改动原因。
- 仓库维护者会定期检查链接有效性和资源可用性，但不承诺对每个页面进行持续维护。
- 对于长期无人维护或内容过时的页面，可能会被标记或移除。
